Graph to highlight function of `same vertex closed` end strategy

### What to look for
When using the end strategy `same vertex closed`, when both searches closed the same vertex and algorithm searches for vertex with the lowest sum, it isnt the vertex that was closed in both searches. Thus, the shortest path doesnt lead through the vertex that was closed in both searches.
